# UnityProject
Small Unity project mainly made to learn Unity. 
 The idea was to make a very little horror game that eventually turned into a platformer.

# Features
The level was build with ProBuilder. It has interactive doors, a moving platform and a moving obstacle. 
I also used a terrain for the end.
For the character movement and model I used Unity Starter Assets third-person controller.
